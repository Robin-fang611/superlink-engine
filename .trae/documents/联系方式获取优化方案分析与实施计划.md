# 联系方式获取优化与精准人员触达方案 (全面升级版)

我已深度阅读最新的《联系方式获取优化方案》。该方案配合您提出的“LinkedIn 关键人触达”功能，将使系统从单纯的“搜公司”进化为“直接找对人”。

## 一、 核心功能可行性分析

### 1. 深度邮箱挖掘 (可行性：极高)
- **技术点**：通过 `site:domain + "@"` 搜索和官网多层级爬取（Contact/About 页面）。
- **优势**：能挖掘出大量隐藏在 PDF 或深层网页中的公开邮箱。

### 2. 关键决策人发现 (可行性：高 - 侧向搜索)
- **技术点**：利用 Google 搜索 LinkedIn 个人主页（如 `site:linkedin.com/in "公司名" "Purchasing"`）。
- **优势**：绕过 LinkedIn 的爬虫封锁，直接获取目标人员姓名和职位。

### 3. 邮箱猜测与智能验证 (可行性：中高)
- **技术点**：基于 `姓名 + 域名` 生成多种组合，并通过我们现有的 SMTP 握手逻辑验证。
- **风险**：需严格控制频率，避免发件服务器 IP 被标记为扫描行为。

### 4. 第三方 API 兜底 (可行性：极高 - 需成本)
- **技术点**：集成 Hunter.io API。
- **优势**：这是目前业界获取企业人员邮箱最稳健、准确率最高的方式。
