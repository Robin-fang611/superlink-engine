                    SuperLink Data Engine - 快速上手指南
==============================================================================

欢迎使用 SuperLink 智能化商业线索挖掘系统！请按照以下三步操作快速启动：

第一步：环境初始化
------------------
双击运行「0_Setup_Environment.bat」。
脚本将自动检查 Python 环境、安装必要的依赖包，并为您生成 .env 配置文件模板。

第二步：配置 API 密钥
------------------
1. 用记事本打开项目根目录下的「.env」文件。
2. 填写您的 API Key：
   - SERPER_API_KEY: 从 https://serper.dev 获取 (用于搜索)
   - ZHIPUAI_API_KEY: 从 https://open.bigmodel.cn 获取 (用于 AI 提取)
3. (可选) 修改 APP_PASSWORD 设置网页登录密码。

第三步：启动引擎
------------------
双击运行「1_Start_Factory.bat」。
等待终端显示「🚀 SuperLink 引擎已就绪！」。
- 本机访问：http://localhost:3000
- 局域网访问：根据终端显示的 IP 地址进行访问。

常见问题 (FAQ):
------------------
1. 无法访问网页？
   - 请确保 3000 端口未被占用。
   - 检查防火墙是否允许 Python.exe 访问网络。
2. 搜索没有结果？
   - 检查 .env 中的 API Key 是否正确。
   - 检查网络是否需要代理（可在 .env 中配置 USE_PROXY）。
