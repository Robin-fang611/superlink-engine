# SuperLink Data Engine 产品功能文档

## 1. 产品概述
**SuperLink Data Engine** 是一款专为 B2B 国际贸易场景打造的智能化商业线索挖掘系统。系统通过集成 Google 实时搜索 API 与大语言模型（LLM）的智能分析能力，能够自动化地从海量互联网信息中精准筛选、提取并验证目标客户（重点关注 SME 中小企业）的业务信息，显著提升市场拓展效率。

---

## 🚀 核心功能模块

### 1. 智能搜索引擎 (Searcher)
*   **多维度搜索**：支持欧美物流商、进口商、中国货代同行、出口工厂等多种搜索策略。
*   **全球裂变**：Ultra 增强模式内置 50+ 全球商贸城市库，自动进行关键词交叉裂变。
*   **异步加速**：支持多线程/异步并行搜索，数据采集效率提升 10 倍以上。

### 2. 精准验证引擎 (Verifier) - NEW!
*   **邮箱三重验证**：自动进行格式校验、MX 记录检查和 SMTP 模拟握手，剔除无效邮箱。
*   **官网深度抓取**：自动访问企业官网，分析业务关键词以确保线索精准度。

### 3. 自动化营销引擎 (EmailSender) - NEW!
*   **个性化触达**：支持响应式 HTML 邮件模板，自动渲染公司名、负责人等动态字段。
*   **频率安全保护**：内置发送间隔和限速机制，最大程度避免邮件被判定为垃圾邮件。

### 4. 客户反馈中心 (Feedback Analysis) - NEW!
*   **自动同步回复**：通过 IMAP 协议自动抓取收件箱中的客户回复。
*   **AI 意向识别**：调用 GLM-4 模型自动分析回复内容，将其分类为“高意向”、“咨询”、“拒绝”等。
*   **营销看板**：直观展示营销转化率和客户意向分布图。

## 3. 业务逻辑流程

```mermaid
graph TD
    A[用户输入: 关键词 + 任务名] --> B{身份验证}
    B -->|通过| C{选择模式}
    C -->|单次/批量/工厂| D[调用 Serper API]
    D -->|原始 JSON| E[AI Processor (GLM-4)]
    E -->|JSON 清洗与提取| F{数据审计}
    F -->|有效 SME| G[四重去重检查]
    G -->|新数据| H[实时写入 CSV]
    H --> I[网页实时预览/下载]
```

## 4. 技术规格与部署

*   **核心引擎**：Python 3.10+, Streamlit, Pandas, ZhipuAI SDK.
*   **网络适配**：
    *   **灵活代理**：支持通过环境变量快速切换代理状态（针对不同部署环境）。
    *   **云端适配**：全面适配 Streamlit Cloud 部署环境，支持 Secrets 密钥管理。
*   **安全性**：支持 SSL/TLS 访问（视部署环境而定）及网页端密码锁。
