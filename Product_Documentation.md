# SuperLink Data Engine 产品功能文档

## 1. 产品概述
**SuperLink Data Engine** 是一款专为 B2B 国际贸易场景打造的智能化商业线索挖掘系统。系统通过集成 Google 实时搜索 API 与大语言模型（LLM）的智能分析能力，能够自动化地从海量互联网信息中精准筛选、提取并验证目标客户（重点关注 SME 中小企业）的业务信息，显著提升市场拓展效率。

---

## 2. 核心功能架构

### 2.1 多维搜索策略 (Search Engine)
系统预置了经过验证的搜索指令模板（Prompt Engineering），支持针对不同贸易角色的精准挖掘：

*   **模块 1: 欧美物流商 (Logistics Provider)**
    *   **目标**：寻找 USA/Europe 本土的货运代理、清关行、仓储服务商。
*   **模块 2: 欧美进口商 (Importer/Distributor)**
    *   **目标**：挖掘 USA/Europe 当地的进口商、批发商、品牌商。
*   **模块 3: 中国货代同行 (China Forwarder)**
    *   **目标**：搜索中国本土从事国际物流的同行企业。
*   **模块 4: 中国出口工厂 (China Exporter)**
    *   **目标**：定位中国本土的生产商、OEM 工厂、外贸出口企业。

### 2.2 智能数据处理流水线 (AI Processor)
系统内置基于 ZhipuAI (GLM-4) 的智能处理引擎，具备高度鲁棒性：

1.  **非结构化数据提取**：智能识别：`公司名称`、`注册国家/城市`、`关键联系人`、`公开邮箱`、`公开电话`、`业务范围`。
2.  **严苛审计与过滤 (Data Audit)**：
    *   **SME 聚焦**：自动剔除年营收 > 1亿美元的大型企业，专注于中小企业。
    *   **无效源清洗**：排除新闻、政府网站、百科等非业务官网数据。
3.  **智能去重 (Deduplication)**：基于 URL、公司名、邮箱和电话的四重指纹比对，确保线索库唯一性。
4.  **鲁棒解析**：优化了 JSON 清洗逻辑，可自动处理 AI 返回的 Markdown 标记及冗余文本，确保数据提取稳定性。

### 2.3 自动化工厂模式 (Automation Factory)
针对大规模数据采集需求的“无人值守”解决方案：

*   **全球城市轮询**：内置 50+ 欧美核心商业城市库。
*   **断点续传**：任务中断重启后自动跳过已完成城市，节省 API 消耗。
*   **可视化监控**：实时展示采集进度条、已完成城市计数及动态日志。

### 2.4 网页管理控制台 (Web Console)
基于 Streamlit 打造的现代化交互界面，支持远程访问与多端协同：

*   **🚀 任务执行页**：支持单次、批量及工厂模式的一键启动。
*   **📂 历史与导出 (New)**：
    *   **在线预览**：直接在网页查看所有历史生成的 CSV 数据。
    *   **一键下载**：内置下载按钮，支持将线索列表即时保存到本地 Excel。
    *   **文件管理**：支持在线清理不再需要的历史文件。
*   **🔒 访问安全 (New)**：集成简易身份验证系统，支持通过 `APP_PASSWORD` 进行网页访问保护。

## 3. 业务逻辑流程

```mermaid
graph TD
    A[用户输入: 关键词 + 任务名] --> B{身份验证}
    B -->|通过| C{选择模式}
    C -->|单次/批量/工厂| D[调用 Serper API]
    D -->|原始 JSON| E[AI Processor (GLM-4)]
    E -->|JSON 清洗与提取| F{数据审计}
    F -->|有效 SME| G[四重去重检查]
    G -->|新数据| H[实时写入 CSV]
    H --> I[网页实时预览/下载]
```

## 4. 技术规格与部署

*   **核心引擎**：Python 3.10+, Streamlit, Pandas, ZhipuAI SDK.
*   **网络适配**：
    *   **灵活代理**：支持通过环境变量快速切换代理状态（针对不同部署环境）。
    *   **云端适配**：全面适配 Streamlit Cloud 部署环境，支持 Secrets 密钥管理。
*   **安全性**：支持 SSL/TLS 访问（视部署环境而定）及网页端密码锁。
