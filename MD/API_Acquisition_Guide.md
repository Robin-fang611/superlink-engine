# SuperLink Data Engine - API 获取与配置全攻略

为了实现“个人邮箱线索增强计划”，我们需要集成几个关键的第三方工具。本指南将带您一步步完成各平台的注册、API 获取及项目配置。

---

## 1. Serper.dev (核心：Google 搜索 API)
**作用**：提供极速的 Google 搜索结果抓取，支持高并发并行查询，是系统线索来源的基石。

### **详细获取步骤**：
1. **访问官网**：打开 [https://serper.dev/](https://serper.dev/)。
2. **注册账号**：点击右上角 **"Sign Up"**。可以使用 Google 账号一键登录，或使用常用邮箱注册。
3. **初始赠送**：新账号注册成功后，通常会自动获得 **2500 次免费搜索额度**（Credits）。
4. **获取 API Key**：
   - 登录后，在左侧导航栏点击 **"API Key"**。
   - 您会看到一个名为 **"Default API Key"** 的长字符串，点击右侧的 **复制图标**。
5. **监控用量**：在 **"Dashboard"** 页面可以实时查看剩余额度。

**配置项**：`SERPER_API_KEY`

---

## 2. Zhipu AI / 智谱清言 (核心：AI 语义分析)
**作用**：利用 GLM-4 模型对搜索到的杂乱信息进行结构化清洗、意向分类以及 JSON 提取。

### **详细获取步骤**：
1. **访问官网**：打开 [智谱 AI 开放平台](https://open.bigmodel.cn/)。
2. **注册与登录**：支持手机号验证码快速登录。
3. **实名认证（必须）**：
   - 为了正常使用 API，通常需要完成实名认证（个人认证即可）。
   - 点击右上角头像 -> **"个人中心"** -> **"实名认证"**。
4. **获取 API Key**：
   - 导航至顶部菜单的 **"控制台"**。
   - 在左侧菜单点击 **"API密钥"**。
   - 点击 **"添加新密钥"**，并为其命名（如 "SuperLink"）。
   - 复制生成的 **API Key**（格式通常为 `xxx.xxx`）。

**配置项**：`ZHIPUAI_API_KEY`

---

## 3. Apollo.io (增强：高精度 B2B 数据库)
**作用**：全球领先的 B2B 数据库，可直接获取精准的决策人姓名、领英 URL 和公司背景，配合猜测算法效果极佳。

### **详细获取步骤**：
1. **访问官网**：打开 [https://www.apollo.io/](https://www.apollo.io/)。
2. **注册账号**：点击 **"Sign Up for Free"**。建议使用**企业邮箱**注册（如 `name@company.com`），免费配额通常比个人邮箱多。
3. **进入设置**：
   - 登录后，点击左下角的 **"Settings"**（齿轮图标）。
4. **定位 API**：
   - 在左侧菜单栏找到 **"Integrations"** 下方的 **"API"**。
5. **创建密钥**：
   - 点击 **"Create New Key"**。
   - 输入名称（如 "SuperLink_Integration"）。
   - 设置权限（默认全选即可）。
   - 点击 **"Create API Key"** 并立即**保存/复制**，因为它只会出现一次。

**配置项**：`APOLLO_API_KEY`

---

## 4. Snov.io (备选：自动化领英线索挖掘)
**作用**：专注领英生态的邮箱抓取，能够将领英个人主页转化为真实的个人邮箱。

### **详细获取步骤**：
1. **访问官网**：打开 [https://snov.io/](https://snov.io/)。
2. **注册账号**：点击 **"Sign Up"** 完成注册。
3. **获取 API 凭证**：
   - 登录后，点击右上角头像 -> **"Account Settings"**。
   - 在左侧菜单中找到并点击 **"API"**。
4. **复制关键参数**：
   - 您需要记录两个参数：**API User ID** 和 **API Secret**。
   - 如果没有 Secret，点击 **"Generate"** 生成一个。

**配置项**：`SNOVIO_USER_ID`, `SNOVIO_API_SECRET`

---

## 🚀 如何配置到项目中？

1. **打开项目目录**：在您的电脑上找到 `superlink数据` 文件夹。
2. **定位环境文件**：找到名为 `.env` 的文件（如果没有，请复制 `.env.example` 并重命名为 `.env`）。
3. **填入密钥**：
   ```env
   # 核心 API (必须)
   SERPER_API_KEY=从Serper获取的长字符串
   ZHIPUAI_API_KEY=从智谱获取的xxx.xxx
   
   # 增强 API (推荐，开启深挖模式需用)
   APOLLO_API_KEY=从Apollo获取的Key
   SNOVIO_USER_ID=从Snovio获取的ID
   SNOVIO_API_SECRET=从Snovio获取的Secret
   
   # 代理设置 (保持默认或按需修改)
   HTTP_PROXY=http://127.0.0.1:7897
   HTTPS_PROXY=http://127.0.0.1:7897
   ```
4. **保存并运行**：保存文件后，重新在终端启动系统即可生效。

## 💡 常见问题排查
* **API 报错 "Invalid Key"**：检查复制时是否多带了空格，或 Key 是否已在官网被禁用。
* **余额不足**：大部分工具提供免费试用，如果用量大建议关注官网的付费计划。
* **网络连接超时**：由于这些 API 服务器多在海外，请确保您的 **代理工具（Clash/V2Ray等）** 已开启，且端口号（如 7897）与 `.env` 中一致。
